## Beweise(n)

Terminiert folgendes Programm immer?
```md [|2,3]

  falls n gerade:  n := n / 2
  sonst:           n := (3 * n) + 1
bis n = 1
```

[Halteproblem](https://de.wikipedia.org/wiki/Halteproblem#Illustration) nach Alan Turing
